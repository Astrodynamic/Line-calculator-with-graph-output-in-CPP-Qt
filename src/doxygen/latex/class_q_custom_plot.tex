Returns the item at the pixel position {\itshape pos}. The item type (a \mbox{\hyperlink{class_q_c_p_abstract_item}{QCPAbstract\+Item}} subclass) that shall be taken into consideration can be specified via the template parameter. Items that only consist of single lines (e.\+g. \mbox{\hyperlink{class_q_c_p_item_line}{QCPItem\+Line}} or \mbox{\hyperlink{class_q_c_p_item_curve}{QCPItem\+Curve}}) have a tolerance band around them, see set\+Selection\+Tolerance. If {\itshape only\+Selectable} is true, only items that are selectable (QCPAbstract\+Item\+::set\+Selectable) are considered.

If there is no item at {\itshape pos}, returns {\ttfamily nullptr}.
