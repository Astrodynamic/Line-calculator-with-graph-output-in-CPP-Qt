Returns the pixel position on the widget surface at which the data point at the given {\itshape index} appears.

Usually this corresponds to the point of data\+Main\+Key/ data\+Main\+Value, in pixel coordinates. However, depending on the plottable, this might be a different apparent position than just a coord-\/to-\/pixel transform of those values. For example, \mbox{\hyperlink{class_q_c_p_bars}{QCPBars}} apparent data values can be shifted depending on their stacking, bar grouping or configured base value.
